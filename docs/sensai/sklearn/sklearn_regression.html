class SkLearnMultiLayerPerceptronVectorRegressionModel(random_state=42, hidden_layer_sizes=100, activation='relu', solver='adam', max_iter=200, **modelArgs)[source]¶

Bases: sensai.sklearn.sklearn_base.AbstractSkLearnMultiDimVectorRegressionModel

__init__(random_state=42, hidden_layer_sizes=100, activation='relu', solver='adam', max_iter=200, **modelArgs)¶
Parameters

  • random_state – the random seed

  • hidden_layer_sizes – the sequence of hidden layer sizes

  • activation – {‘identity’, ‘logistic’, ‘tanh’, ‘relu’} the activation function to use for hidden layers (the one used for the output layer is always ‘identity’)

  • solver – {‘lbfgs’, ‘sgd’, ‘adam’} the name of the solver to apply

  • max_iter – the number of iterations (gradient steps for L-BFGS, epochs for other solvers)

  • modelArgs – additional arguments to pass on to MLPRegressor, see https://scikit-learn.org/stable/modules/generated/sklearn.neural_network.MLPClassifier.html
